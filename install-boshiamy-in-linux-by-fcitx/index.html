在Ubuntu使用fcitx安裝嘸蝦米

發表於 2014-06-16 分類於 記錄 文章字數： 994

之前用過gcin, scim等輸入工具，發現對嘸蝦米的支援十分有限。fcitx預設即有嘸蝦米碼表，較為方便，安裝也比較簡單

使用 Ubuntu 12.04 32bit 測試 (後有16.04 和 22.04 方法)

安裝fcitx輸入工具

sudo apt-get install im-switch fcitx

安裝嘸蝦米於fcitx

sudo apt-get install fcitx-boshiamy

設為預設值

使用im-switch將fcitx設為預設輸入法
sudo im-switch -s fcitx

調整fcitx設定

fcitx設定 → 附加組件配置 → 碼表 → boshiamy.config

模糊鍵

『使用模糊鍵』不勾
此鍵預設被打勾，並定為z鍵，導致所有含z碼的字都出不來…

使用符號

要用嘸蝦米預設符號的話，「忽略標點」打勾

拼音查蝦米碼

在拼音鍵打勾，最好選 `（Esc下的那鍵），才不影響輸入，然後用 -, = 兩鍵翻頁。
（至於輸入的字要查發音則是屬於另一個功能，在剛按完空白鍵輸入後，就按Ctrl+Alt+E，即可看見發音，但缺點是沒有四聲）

Reinstall: Ubuntu 16.04 64bit

sudo apt-get install -y fcitx fcitx-table-boshiamy
im-config # choose fcitx
fcitx-configtool # choose boshiamy
# restart or relogin

完成後用Ctrl+Space及Ctrl+Shift切換輸入法即可使用

Reinstall: Ubuntu 22.04 64bit

sudo apt install -y fcitx5 fcitx5-chinese-addons
im-config # choose fcitx5
# 下載碼表並安裝
wget https://eu.mirror.archlinuxarm.org/aarch64/community/fcitx5-table-extra-5.0.9-1-any.pkg.tar.xz # 請自行尋找最新版本
unxz fcitx5-table-extra-5.0.9-1-any.pkg.tar.xz
tar xvf fcitx5-table-extra-5.0.9-1-any.pkg.tar
sudo cp -r usr/share /usr/
fcitx5-configtool # choose boshiamy
# restart or relogin
