簡報製作軟體比較

發表於 2022-03-22 分類於 心得 文章字數： 501
此文介紹常用的簡報製作方式及比較

分類

  1. 編輯軟體
  2. Latex
  3. Markdown
  4. Javascript
  5. 其他

編輯軟體

  • 優點
    • 編輯方便
    • 功能齊全
    • 模版多
  • 缺點
    • 手動排版
    • 需要使用特定軟體才能編輯

軟體

  • PowerPoint
  • Google Slides
  • Prezi

Latex

  • 優點
    • 自訂性高
    • 寫公式、程式碼方便
  • 缺點
    • 指令過於複雜
    • 需要設定開發環境
  • 模版
    • Beamer
      • BeamerTaiwan

Markdown

  • 優點
    • 原始文字易於保存
    • 語法較Latex簡單
  • 缺點
    • 自訂性比Latex差
    • 難製作動畫

軟體

  • Marp
    • 可以寫html tag
    • 簡易的css支援
    • 動畫功能較少
    • 教學

Javascript

  • 用js擴充功能
  • 用css排版

軟體

  • Reveal.js
    • 支援Markdown語法
    • 教學:Reveal.js配合Markdown
  • Slidev
    • 支援Markdown語法
    • 支援類Vue語法
  • Impress.js
    • 類似Prezi
  • Deck.js
  • Ascensor
  • LandSlide

其他

用遊戲引擎做簡報

  • Unity
  • 可以將動畫和特效做到非常好，只是過於麻煩

結論

  • 使用純文字(即Markdown)儲存簡報，較沒有相容性的問題，版本控制也容易
    • 但排版較困難
  • css是目前排版的最好方法
  • 所以最好的方法為Markdown+html
  • 推薦使用
    • Marp
    • Reveal.js
    • Slidev
